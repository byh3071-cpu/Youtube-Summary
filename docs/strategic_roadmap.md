# [전략 로드맵] 포커스 피드 스케일업 및 고도화 전략 (2025-2026)

이 로드맵은 '포커스 피드'를 단순한 피드 리더에서 **'AI 1인 기업가를 위한 지능형 지식 엔진'**으로 진화시키기 위한 단계별 전략을 제시합니다.

---

## 🏗️ 4대 핵심 전략 기둥 (Pillars)

1.  **지능형 통찰 (Intelligence)**: 단순 요약을 넘어 트렌드를 감지하고 질문에 답하는 능력을 배양합니다.
2.  **생태계 확장 (Ecosystem)**: 유투브/RSS를 넘어 트위터(X), Reddit, 커뮤니티 등 더 빠르고 다양한 소스를 통합합니다.
3.  **워크플로우 통합 (Workflow)**: 습득한 지식이 생각으로만 머물지 않고, 실제 태스크(Task)와 노트(Note)로 이어지게 합니다.
4.  **플랫폼 견고화 (Platform)**: 언제 어디서나 끊김 없이, 수천 개의 데이터를 지연 없이 처리할 수 있는 성능을 확보합니다.

---

## 📅 단계별 실행 계획 (Phases)

### Phase 1: 지능적 필터링 및 개인화 (Short-term)
*목표: 사용자의 'My Focus'를 모든 기능의 중심으로 배치*
-   **AI 트렌드 레이더**: 수집된 피드 전체에서 공통 키워드를 추출하여 '오늘의 핫 토픽' 자동 생성.
-   **스마트 태깅**: AI가 피드 내용을 분석해 `#ClaudeCode`, `#SaaS마케팅` 등 태그를 자동 부여하고 필터링 지원.
-   **알림 고도화**: 설정한 핵심 키워드가 포함된 콘텐츠가 올라오면 즉시 브라우저 푸시 알림.

### Phase 2: 대화형 지식 탐색 (Medium-term)
*목표: 피드 데이터와의 직접적인 상호작용*
-   **피드 기반 RAG (Chat)**: 내가 구독한 채널들의 내용을 기반으로 답변하는 전용 챗봇 구현. "최근 올라온 AI 에이전트 사례들 다 정리해줘" 같은 요청 처리.
-   **커뮤니티 소스 연동**: Reddit의 기술 서브레딧, X(Twitter)의 특정 리스트를 피드에 통합.
-   **AI 뉴스 앵커 (TTS)**: 아침마다 상위 3개 뉴스를 음성 브리핑으로 생성해주는 '데일리 팟캐스트' 기능.

### Phase 3: 실행 지향적 워크플로우 (Integration)
*목표: '읽기'에서 '실행'으로의 전환*
-   **외부 서비스 연동**: '인사이트 정리' 결과를 클릭 한 번으로 Notion, Obsidian, Slack, Todoist 등으로 전송.
-   **퍼스널 대시보드**: 나의 학습 진행도, 관심 분야 추이, 이번 주에 뽑아낸 액션 아이템 리스트를 한눈에 보는 대시보드.
-   **유사 콘텐츠 추천**: 내가 북마크한 콘텐츠와 유사한 새로운 채널/블로그 자동 추천.

### Phase 4: 스케일업 및 클라우드 동기화 (Scalability)
*목표: 매끄러운 사용자 경험과 데이터 처리 최적화*
-   **Supabase 전체 동기화**: 모든 설정, 북마크, 시청 기록을 실시간 DB 동기화하여 기기 간 경계 제거.
-   **PWA 및 오프라인 모드**: 네트워크가 불안정한 환경에서도 읽어둔 피드와 요약을 볼 수 있는 오프라인 경험 최적화.
-   **벡터 DB 도입**: 수천 개의 피드 내용을 효율적으로 검색(Semantic Search)하기 위한 벡터 인덱싱 도입.

---

## 📈 스케일업을 위한 기술적 제언

### 1. 성능 최적화 (Scaling performance)
-   **Incremental Static Regeneration (ISR)**: RSS/YouTube API 호출을 백그라운드 워커로 분리하고, Next.js의 ISR을 적극 활용해 사용자 체감 속도 극대화.
-   **Edge Computing**: 사용자와 가까운 위치에서 피드 병합 로직을 처리하여 대기 시간(Latency) 단축.

### 2. 비용 관리 (Scaling cost-effectively)
-   **Caching Layer**: Gemini API 호출 비용 절감을 위해 동일 영상/글에 대한 요약은 Supabase에 영구 캐싱 (이미 구현 중인 부분을 모든 데이터로 확대).
-   **Token Optimization**: 요약 시 불필요한 데이터를 제외하고 핵심 컨텍스트만 AI에 전달하여 토큰 소모 최적화.
