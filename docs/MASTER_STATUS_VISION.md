# [마스터 보고서] 포커스 피드: 현황 및 비전 통합 가이드

이 문서는 '포커스 피드' 프로젝트의 현재 진행 상황을 점검하고, 앞으로 나아갈 전체적인 방향성과 구체적인 실행 계획을 한눈에 볼 수 있도록 정리한 마스터 가이드입니다.

---

## 1. 현재 진행 상황 (Current Status)

기술적 부채를 해결하고 고도화를 위한 완벽한 기반을 마련했습니다.

- **✅ 코드 리팩토링 완료**: AI 프롬프트 중앙 관리(`prompts.ts`), Supabase 타입 안정성 강화(`getTypedTable`), Gemini 모델 최적화 완료.
- **✅ 개발 가이드 수립**: 커서(Cursor)가 고효율로 작업할 수 있는 기술 지침서(`docs/cursor_implementation_guide.md`) 작성 완료.
- **✅ 전략 리포트 구축**: 비즈니스 전략, 비용 분석, 로드맵 등 6종의 전문 리포트를 `docs/` 폴더에 배치 완료.

---

## 2. 큰 그림: 방향성 및 비전 (Direction & Vision)

> **"정보 홍수 속에서 내 비즈니스에 필요한 정수만 뽑아내어 실행으로 연결하는 'AI 지식 엔진'"**

단순한 유튜브 요약 앱이 아니라, AI 1인 기업가가 매일 아침 가장 먼저 켜서 **세상의 변화를 읽고 즉각적인 액션 아이템을 얻는 OS**로 진화합니다.

---

## 3. 아이디어 및 고도화 계획 (Phased Roadmap)

| 단계 | 명칭 | 핵심 가치 | 핵심 기능 |
| :--- | :--- | :--- | :--- |
| **Phase 1** | **Intelligence** | 개인화된 통찰 | 실시간 트렌드 레이더, 스마트 태깅, 키워드 알림 |
| **Phase 2** | **Interactive** | 대화형 지식 탐색 | 피드 기반 RAG 챗봇, AI 뉴스 오디오 브리핑(TTS) |
| **Phase 3** | **Workflow** | 실행 자동화 | Notion/Todoist 연동, 원클릭 인사이트 내보내기 |
| **Phase 4** | **Platform** | 확장성 및 안정성 | 벡터 DB 도입, PWA 오프라인 모드, 기기간 동기화 |

---

## 4. 우선순위 및 체크리스트 (Priorities & Checklist)

앞으로 커서(Cursor)와 함께 수행할 최우선 과제들입니다.

### 🚩 [우선순위 1] 지능형 트렌드 레이더 (Phase 1)
- [ ] 최근 24시간 내 수집된 피드 분석 API 구축
- [ ] 상위 5개 핵심 트렌드 키워드 추출 로직 (Gemini Flash)
- [ ] 대시보드 상단 트렌드 바(Bar) UI 구현

### 🚩 [우선순위 2] 피드 데이터 벡터화 및 라이브러리 (Phase 2 준비)
- [ ] Supabase `pgvector` 활성화 및 엠베딩 저장 구조 설계
- [ ] 과거 요약 내역 시맨틱 검색(Semantic Search) 기능 테스트

### 🚩 [우선순위 3] 워크플로우 연동 (Phase 3 준비)
- [ ] 노션(Notion) API 연동 테스트 및 템플릿 설계
- [ ] 모바일 환경에서의 오디오 브리핑 경험 개선

## 5. 비용 및 운영 관리 (Operation Management)

- **비용**: 1인 사용 시 **월 $0(무료)** 유지 가능. (Vercel + Supabase + Gemini 무료 티어 조합)
- **성능**: 하이퍼 파라미터 튜닝과 캐싱 로직을 통해 API 호출 속도 극대화 유지.
