# 라디오 플레이어 버튼 아이콘 (5개)

아래 **폴더**에 이미지를 넣을 때 **파일명**을 아래와 동일하게 맞춰 주세요.

---

## 폴더 위치

```
프로젝트/public/images/icons/
```

---

## 5개 파일 (현재 적용된 이름)

| 번호 | 파일명 | 용도 |
|------|--------|------|
| 1 | **Close Player.png** | 플레이어 닫기 |
| 2 | **Feed_List1.png** | 재생 대기열 |
| 3 | **Watch mini-video.png** | 미니 영상 켜기/끄기 |
| 4 | **AI_summary1.png** | AI 요약(가사) 보기 |
| 5 | **View the full screen.png** | 전체 화면 영상 |

코드에서 사용하는 경로는 공백을 `%20`으로 인코딩합니다 (예: `Close%20Player.png`).

---

## 코드 매핑 (ThemeIcon name)

- `close_player` → Close Player.png  
- `feed_list1` → Feed_List1.png  
- `watch_mini_video` → Watch mini-video.png  
- `ai_summary1` → AI_summary1.png  
- `view_fullscreen` → View the full screen.png  

이미지 업데이트 후 위 파일명만 맞추면 라디오 푸터에 자동 반영됩니다.
