# 조코딩 해커톤 공모전 · 수익/사업화 관점 (작성 참고용)

공모전 서류에 "수익 가능성" 또는 "사업화 전망"을 넣을 때 아래 문단을 그대로 또는 요약해서 사용할 수 있습니다.

---

## 1. 한 줄 요약 (수익 관점)

> **"개인/팀의 시간을 줄여주는 AI 큐레이션 도구로, 프리미엄 기능·팀/기업용 구독으로 수익화할 수 있는 구조입니다."**

---

## 2. 수익화 방향 (공모전용 3가지)

| 방향 | 설명 | 수익 형태 |
|------|------|-----------|
| **프리미엄 개인** | AI 요약·인사이트 무제한, 목표 기반 브리핑, 챕터/구간 요약 등 고급 AI 기능을 유료화 | 월 구독 (예: 소액 프리미엄) |
| **팀/기업용** | "팀이 이번 주 꼭 볼 영상 3개" 공유, 팀 북마크·브리핑, 슬랙/노션 연동 | 팀 단위 월 구독 또는 연간 라이선스 |
| **B2B·콘텐츠 제작자** | 크리에이터가 자신의 영상 요약/챕터 자동 생성, 공유용 카드 생성 등 도구 제공 | 채널/크리에이터 단위 구독 또는 API 사용량 과금 |

---

## 3. 구체적 수익화 시나리오

### 3-1. 개인용 (Free vs Pro)

| 구분 | 무료(Free) | 유료(Pro, 가칭) |
|------|------------|------------------|
| **AI 3줄 요약** | 일 N회 제한 (예: 5~10회/일) | 무제한 |
| **인사이트** | 비활성 또는 1회/영상 | 무제한 |
| **목표 기반 브리핑(My Focus)** | 주 1회 또는 상위 3개만 | 매일 + 상위 10개·상세 이유 |
| **챕터/구간 요약** | 미제공 | 구간별 AI 챕터·한 구간 요약 |
| **라디오 큐** | 제한 없음(유지) | 동일 + 재생 목록 내보내기 등 |
| **가격 가이드** | — | 월 4,900원~9,900원 수준 검토 (참고용) |

→ **타깃**: 학습·자기계발·업무에 유튜브를 많이 보는 1인 창업자, 직장인, 대학원생. “시간 아끼기”에 월 만 원 이하 결제 수용 가능한 층.

### 3-2. 팀/기업용 (Team)

| 기능 | 내용 |
|------|------|
| **팀 브리핑** | 팀 목표/키워드 입력 → “이번 주 팀이 볼 영상 N개” + 이유·액션 제안. 매주 자동 갱신 가능. |
| **공유 북마크** | 팀 전용 북마크 폴더·태그. 신규 입사자 온보딩, 스터디 그룹 자료 공유. |
| **연동** | 슬랙 봇(“오늘 추천 영상”), 노션 DB 연동(북마크·요약 자동 정리). |
| **가격 가이드** | 팀당 월 29,000원~99,000원(인원수·기능에 따라 단계별). 연간 결제 시 할인. |

→ **타깃**: 스타트업, 스터디 그룹, 마케팅/기획 팀. “함께 볼 콘텐츠 정리·공유”에 예산 쓰는 조직.

### 3-3. B2B·크리에이터 (Creator/API)

| 기능 | 내용 |
|------|------|
| **채널 단위 요약/챕터** | 업로드 영상에 대해 AI 요약·챕터 제목 자동 생성 → 커뮤니티/블로그/뉴스레터에 붙여넣기. |
| **공유용 카드 생성** | 요약 문장 기반 OG 이미지·썸네일 대안 생성(이미지 API 연동 시). |
| **API** | 요약·챕터 생성 API를 미디어사·교육 플랫폼에 제공, 호출당 또는 월 구독 과금. |
| **가격 가이드** | 채널당 월 구독 또는 API 호출당 과금(예: 1,000건당 N원). |

→ **타깃**: 유튜버·강사·미디어사. “영상 정리·공유 자동화”에 비용 지불 가능한 주체.

## 4. 공모전 서류에 넣을 때 추천 문단 (복사용)

**옵션 A – 짧게 (2~3문장)**  
> Focus Feed는 유튜브·RSS를 한 곳에서 보고, AI로 요약·추천까지 받는 개인 큐레이션 도구입니다. 수익은 (1) 개인용 프리미엄(무제한 AI 요약·브리핑), (2) 팀/기업용 구독(팀 브리핑·공유 북마크)으로 확장할 수 있고, 장기적으로는 크리에이터용 요약/챕터 도구로 B2B까지 고려하고 있습니다.

**옵션 B – 구체적으로 (공모전 본문용)**  
> **수익화 전망**  
> - **개인**: 무료는 일일 AI 요약 횟수 제한(예: 5~10회), 유료(Pro)는 요약·인사이트·목표 기반 브리핑 무제한과 챕터/구간 요약을 제공. 월 4,900~9,900원 수준의 소액 구독으로 “시간 절약·학습 효율”에 결제하는 1인 창업자·직장인·학습자를 타깃으로 함.  
> - **팀/기업**: 팀 목표 기반 “이번 주 꼭 볼 영상” 브리핑, 팀 북마크·공유 폴더, 슬랙/노션 연동으로 지식 공유 도구로 판매. 팀당 월 29,000~99,000원 수준, 연간 결제 시 할인으로 스타트업·스터디·마케팅 팀 수요를 노림.  
> - **B2B·크리에이터**: 채널별 요약·챕터 자동 생성, 공유용 카드 생성, API 제공으로 크리에이터·미디어사에 호출당 또는 월 구독 과금.  
> 사용자 가치는 “볼 영상 고르는 시간 절약 + AI로 핵심만 빠르게 파악”이므로, 시간·학습 효율에 돈을 쓰는 개인과 팀이 결제 전환 후보이며, 단계적으로 개인 → 팀 → B2B로 확장할 수 있는 구조입니다.

**옵션 C – 한 문장만**  
> 개인 프리미엄(무제한 AI)·팀 구독·크리에이터용 도구까지 확장 가능한 수익 구조를 갖춘 개인/팀 큐레이션 앱입니다.

**옵션 D – 단계별 확장 (로드맵 느낌)**  
> 1단계(현재~): 무료 개인 사용으로 사용자 확보·AI 품질 검증.  
> 2단계: 일일 요약 제한 도입 + Pro 구독(무제한 요약·브리핑·챕터)으로 개인 수익화.  
> 3단계: 팀 브리핑·공유 북마크·연동 기능으로 팀/기업 구독 런칭.  
> 4단계: 크리에이터·미디어사 대상 요약/챕터 API·공유 카드로 B2B 확장.

## 5. 주의할 점 (해커톤 공모전)

- **과장하지 않기**: "수익 보장"이 아니라 "수익화 방향·가능성"으로 쓰는 것이 안전합니다.
- **기술/아이디어가 주인공**: 수익은 "지속 가능성·확장성"을 보여주는 보조 논리로 두는 편이 좋습니다.
- **심사 기준 확인**: 공모전이 "기술·창의성" 위주면 수익 문단은 짧게, "사업성·스케일"을 보면 위 3가지 방향을 조금 더 쓰면 됩니다.
